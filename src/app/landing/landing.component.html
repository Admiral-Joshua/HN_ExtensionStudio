Extension Studio

The Extension Studio is a quick and easy way to get started with your Hacknet Extensions.
Designed from the ground up to utilize a RESTful API and a strong relational database, to provide you with an easy start-up.
With linking support between Missions, Emails, Board Postings, Actions, Computers and their security and much more!
Whilst we don't have any intentions of completelyreplacing good ol' fashioned coding (though it's in theory possible) we hope to make your life easier a bit easier with getting your extension started.



Site Status: Partially Available -> Some features, such as: action sets, Themes and compilation to XML are still in development, but you can now largely create your own extension.
Think you can do better than us?

Sadly we're still working on our UI editor but we're excited to tell you about our server back-end.
Whilst perhaps not as exciting for some we've really made an effort to utilize OpenAPI v3.
This provides us with a framework that we can publicly make available for anyone to develop a user interface for! If you're not familiar with RESTful API services then we can't really help you just yet!

But if you are, and want to get started, check out the docs here!
